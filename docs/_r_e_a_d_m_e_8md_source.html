1 # LEAC - Library Evolutionary Algorithms for Clustering <img align="right" width="100" height="100" src="leac_logo.png">
2 
3 Library Evolutionary Algorithms for Clustering (**LEAC**) is a library for the implementation
4 of evolutionary and genetic algorithms to solve the problem of *partition clustering*.
5 The tools is free and open source, under the GNU General Public
6 [GPLv3 license](https://www.gnu.org/licenses/gpl-3.0.en.html)
7 [\[License file\]](../../LICENSE),
8 for Windows&reg;, GNU/Linux&reg; and Mac OS X&reg;.
9 For more details see <a href="https://github.com/kdis-lab/leac/tree/master/doc/leac-userManual.pdf">user
10 manual</a>.
11 
12 Clustering is useful in several exploratory *pattern-analysis*,
13 *grouping*, *decision-making*, and *machine-learning situations*,
14 including *data mining*, *document retrieval*, *image segmentation*,
15 and *pattern classification* <a href="http://doi.acm.org/10.1145/331499.331504">Jain et al</a>.
16 
17 ![](../master/doc/leac_cluster.svg)
18 
19 LEAC is built in C ++ with the current standards of C++11, C++14.
20 Take advantage of the STL library, through the use of
21 data structures and algorithms, for example,
22 Numerics library of Pseudo-random number generation which is
23 fundamental in the genetic and evolutionary algorithms.
24 An additional feature that LEAC has is a low-level software layer
25 based on Streaming SIMD Extensions (SSE) and in the
26 [OpenBLAS](http://www.openblas.net) library ,in order to
27 increase performance. For now only for Linux x86_64, for future
28 versions it is intended to port to other architectures and
29 increase the number of genetic operators implemented with
30 these software layers
31 
32 LEAC also includes several implementations of evolutionary algorithms for
33 partitional clustering which are based on state-of-art of evolutionary
34 proposals in this area:
