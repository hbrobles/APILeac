Detailed Description

Interface to Streaming SIMD Extensions (SSE) high Performance Level Low Functions.

Interface to Streaming SIMD Extensions (SSE) kernel function.

Functions base in SSE instructions operate on packed double-precision floating-point values contained in XMM registers and on packed integers contained in MMX and XMM registers [24]
